Sobre webci.h

¿Qué es webci?

Webci es una libreria para crear servidores webs y usar C para enviar paginas html o crearlas y tener mas control de las respuestas enviadas(o solicitudes) de un sitio web y enviar respuestas de acorde a ello.

¿De que trata el proyecto?

Webci es un proyecto para practicar el lenguaje C, pero tambien es un desafio hacer una libreria que funcione en C para la web aunque por el momento sea en servidores creados por uno mismo, la libreria se mejora con el pasar del tiempo y aun esta en desarrollo.

Cualquier persona que desee puede cojer el codigo fuente y usarlo , como tambien podra usarlo para funciones en especifico y no para crear servidores que envien paginas webs

¿Cual sera el objetivo principal o final?

El objetivo de Webci sera crear una libreria para usar C en la web y el final de momento sigue igual que el principal.
